You will have the chance to work with 3D printing, Collaborative Industrial Robotic Arms, Artificial Intelligence and much more. If you are a motivated student and willing to quickly learn and deliver results fast, these Internships are made for you!


Data Analysis and Implementation of Machine Learning Pipelines for Real Industry Datasets

Beginner Internship

1st July - 30th September

Machine Learning

Informatics / Electronics Engineering


Exploration of real industry datasets, including tasks for data aggregation, cleaning & wrangling; data visualization; feature extraction & engineering; model selection & evaluation. The complete algorithm should be able to compute and estimate performance indicators, such as identifying machine stops and failures, spot defects causes, or others, depending on the available data. The algorithm will use regression and classification tasks, mixed features (numerical discrete or continuous, categorical, ordinal, word-based, ...), and/or structured/unstructured datasets. Data sources include sensor data, PLC event logs, production machines, historical tabular data, or human-labeled data. Technologies: Numpy, Pandas, Matplotlib, Scipy, Scikit-Learn.

Formulating a tool to aid factory shopfloor decision makers selecting a suitable gamification approach

Beginner Internship

1st July - 30th September

Management/ organizational culture/ digitalization

Informatics / Electronics Engineering


Gamification, widely accepted as “the use of game design elements in non-game contexts”, is often used to increase motivation, engagement and training experiences. Despite its potential, in the manufacturing sector is still unexplored and is frequently integrated as a one size-fits-all solution, disregarding specific user needs and organizational or contextual conditions. The tool developed within this internship should allow the user (factory shopfloor decision maker) to enter data about technologies/devices used in the workplace, motivation and users specific characteristics (i.e. age and technological literacy) and needs. The tool can then generate an appropriate solution, according to the data entered.

Beginner Internship

1st July - 30th September

Gamification within production settings

Multimedia master students interested in starting a career as a researcher

Asset Administration Shell for Distributed Systems developed in DINASORE

Beginner Internship

1st July - 30th September

Software Development

Informatics / Electronics Engineering


The Asset Administration Shell (AAS) is a standardized digital representation that encapsulates comprehensive information about industrial assets, enabling seamless integration, interoperability, and lifecycle management within the context of Industry 4.0. The goal of this internship is to develop a tool that integrates the distributed systems developed using DINASORE with the AAS standard. Function Block Testing and Validation Tool

Beginner Internship

1st July - 30th September

Software Development

Informatics / Electronics Engineering


The test and validation of function blocks (FBs) offers the advantage of ensuring the reliability and correctness of the code, identifying potential bugs or issues early on, and facilitating seamless integration into larger software systems. The internship goal is to develop a tool that performs a set of tests to FBs, identifying mistakes in the their development, such as a wrong number of inputs or an incorrect FB type. Technologies: Python, DINASORE (https://github.com/DIGI2-FEUP/dinasore)

IIoT Power Meter Module

Beginner Internship

1st July - 30th September

Embedded SW/HW Development

Electrical Engineering


Create a DIN module capable of sensing voltage and current and calculate apparent power, reactive power and active power, as well as energy counters. This module shall be capable of reporting its measurements over the network as well as report the meter location and machine where it is installed. In this work you will be able to fully develop a prototype of the power meter and later an industrial version to be applied in a real use-case scenario. Domains: Electronics, Embedded Development, Firmware

Digital Twin Software Module

Beginner Internship

1st July - 30th September

Software Development

Informatics/ Electrical Engineering


Create a distributed software environment for managing the Digital Twins of electrical appliances and machines. On one side this software shall embed a small server that contains a standard description of a machine as well as an endpoint to read/write live or constant machine information. The client side of this software should list all available machines and respective info, and for each digital representation of the machine host a series of predefined dashboards. This side will also embed a DB to store the machine's historical data.
